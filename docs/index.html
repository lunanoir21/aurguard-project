# Why

AUR helpers happily run untrusted build code as your user — and .install scriptlets as root. A typo-squatted package, a hijacked maintainer account, or a single curl … | sh line is all it takes. aurguard puts a review step in front of that.

🧬

Clone-first, TOCTOU-safe

The repo is cloned once, then the same tree is analyzed and built — no swap between what you read and what runs.

🌳

Real bash parsing

A tree-sitter-bash AST sees eval and pipe-into-shell through quoting tricks, comments, and here-docs.

🚫

Never executes

Analysis is purely static. A clean report means “no known-bad shapes,” and nothing in the package has run.

# What it checks

Four analysis passes feed one report: the AST pass, textual heuristics, a YARA-style signature database, and package metadata & history.

🔎

YARA-style signatures

A dependency-free rule engine matches named malware shapes — miners, exfiltration, persistence, defense-evasion — with vetoes that stay quiet on legitimate $pkgdir packaging.

🌐

Source & metadata

Insecure/IP/shortened source URLs, SKIP checksums, untrusted git clone, maintainer age, low votes, staleness, and pkgrel churn.

🔐

Root scriptlets

.install hooks run as root, so they get the full rule set plus a flag for any network access at install time.

📝

Approval ledger

Approve a PKGBUILD once; aurguard surfaces silent upstream edits on the next install via a content digest.

⏿

Scriptable

--json output and exit codes (0/10/20) make it a clean CI gate.

🗣️

Five languages

Interface and findings localized end to end.

# A sample of the rules

Findings are bucketed INFO · WARN · CRITICAL; a package's risk is the highest severity it trips. CRITICAL is blocked by default.

CURL_PIPE_SH REVERSE_SHELL CRYPTO_MINER DISCORD_EXFIL CRED_HARVEST SSH_KEY_INJECT CRON_PERSIST SUID_BIT DESTRUCTIVE SYSTEM_PATH_WRITE UNKNOWN_SOURCE CHECKSUM_SKIP INSTALL_NETWORK URL_SHORTENER INSECURE_FETCH PIP_INDEX_HIJACK

# Install

Linux only — the AUR is Arch-specific.

From source (recommended) 

# needs the Rust toolchain + a C compiler (tree-sitter-bash grammar)
git clone https://github.com/lunanoir21/aurguard-project.git
cd aurguard-project
cargo build --release
./target/release/aurguard --setup

From npm 

npm install -g aurguard

The npm package is a thin launcher with no install script. The prebuilt binary ships as a platform-specific optional dependency, so nothing is downloaded or executed at install time — no postinstall, no remote-code surface, zero runtime dependencies.

From crates.io 

cargo install aurguard
